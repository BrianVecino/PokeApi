# PokéAPI Explorer 🎮

Aplicación desarrollada con Angular 21 que consume la [PokéAPI](https://pokeapi.co/) 
para explorar y buscar información de Pokémon.

## Funcionalidades
- Listado de Pokémon con paginación
- Búsqueda por nombre
- Vista de detalle con stats y tipos
- (agregá lo que tenga tu app)

## Tecnologías
- Angular 21
- TypeScript
- PokéAPI (REST)
- CSS

## Cómo correrlo localmente
bash
git clone https://github.com/BrianVecino/PokeApi
cd PokeApi
npm install
ng serve
